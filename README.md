# ReviewLens

AI-powered sentiment analysis system for Daraz product reviews using:

- Flutter
- FastAPI
- Scikit-learn
- NLP
- Kaggle

## Features

- Real-time review analysis
- Sentiment classification
- Positive/Negative insights
- Data visualization
- Daraz review scraping

## Tech Stack

### Frontend
- Flutter

### Backend
- FastAPI

### AI/ML
- Scikit-learn
- TF-IDF
- Logistic Regression
